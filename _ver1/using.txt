Запуск программ : 

1) Консольная (Shapes\bin\Debug\Shapes.exe) :
а) запустить cmd.exe
б) Запуск программы : Shapes.exe (0 режим), Shapes [N режима, с 0 по 3]
Возможность ввода фигур отсутствует - можно поправить файл shapes.csv

2) GUI (ShapesGUIApp\bin\Debug\ShapesGUIApp.exe)
Все в общем понятно из GUI. Можно указать источник данных (Из файла shapes.csv,
тестовый источник (константы), тестовый большой источник (1 млн. фигур),
ввести фигуры в форме.
